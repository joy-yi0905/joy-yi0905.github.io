markdown

2012-01-01

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

同时，Markdown旨在让作者专心写作，而无需关注网页的排版。

常用的markdown格式主要包含以下这些：

  • 标题、段落、无序列表，有序列表、表格
  • 斜体、加粗、删除线、下划线、小文本、行内代码
  • 链接、邮箱、图片
  • 代码块
  • 多媒体
  • 引用
  • 分割线
  • todolist
